## 项目研究自用。 不接受建议。特此声明，谢谢。
### Install
```
sudo pip install xray-util@git+https://github.com/zerofo-tang/multi-xray@master
```
### Upgrade multi-xray
#### 方法一 (3.10.9.10 之后 版本才支持)
```
sudo pip install -U xray-util@git+https://github.com/zerofo-tang/multi-xray@master
```
#### 方法二
```
xray-util update.sh
```
### Upgrade xray-core
#### 方法一
```
xray-util update
```
#### 方法二
```
xray-util ->(进入菜单) 6
```
